# Ames Housing EDA

## Objective

To analyze the key factors influencing house prices in Ames, Iowa.

## Dataset

Ames Housing dataset (Kaggle)

## Key Findings

- Larger living area significantly increases price.
- Neighborhood creates major pricing differences.
- OverallQual is the strongest driver of SalePrice.

## Techniques Used

- Pandas for data manipulation
- Matplotlib & Seaborn for visualization
- Correlation analysis

## What I Learned

- Structured EDA workflow
- Interpreting correlations
- Handling structured missing data
